Stress Less - VR App to Reduce Stress

Summary

StressLess is a virtual reality app that calms students with different virtual environments and includes features such as breathing exercises and emotional scales for the user to lower stress levels.

Tools Used

  • Created using Android Studio
  • Virtual environments made in Unity
  • Designed in Adobe XD
  • Data from user is stored in Cloud Firestore
