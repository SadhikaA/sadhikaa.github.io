FlameLess - Wildfire Sensor

Summary

Flameless is a wildfire sensor array that detects a fire at its source and relays the necessary information to the fire department. The likelihood of a fire is determined by temperature, humidity, wind speed, wind direction, smoke levels, predicted acres burned based on elevation.

2020 Winners - Dougherty Valley High School

Project

Tools Used

  • Sensor made using Arduino Uno, LoRaWan Shield, DHT11 Temperature and Humidity Sensor, and an MQ-2 Smoke Sensor
  • LoRa Gateway used to relay information
  • Data graphed and presented through Cayenne
  • Machine learning model to predict acreage burned, created through Keras
  • 3D Printed Case using ASA filament, designed through Autodesk Fusion 360
