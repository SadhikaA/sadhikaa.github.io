TOURNAMENT PLAYER RANKING

Summary

When finding players in a tournament and comparing their UTR’s and rank’s, there is no existing program to provide results.

  1. Given tournament ID (from USTA website)
  2. Find either Applicants or Competitors in the tournament (user chooses from a combo box)
  3. Read and save data from table of players into a data structure
  4. Determine the ranking of every player by comparing with USTA database of rankings by division (B12s, B14s, B16s, B18s, G12s, G14s, G16s, G18s)
  5. Add a link for each player to their UTR.

Process & Expansion

  • Console App (tournamentID is user input, prints the information and whether the player is selected through main function, displayed in IDE)
  • Applet (displays a form for user input, print information below, UI designed with Java Swing, results displayed in the Applet)
  • Web Page (displays in browser with a form for user input, prints all players in a table in the website, using JSP to build UI)

Tools Used

  • Java, Maven
  • JSP (Java Server Pages) and Servlets
  • HTMLunit, Apache POI
  • HTML, CSS, and Javascript
